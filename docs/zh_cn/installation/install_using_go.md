# go方式安装

## 环境准备
* golang 1.13+

## 安装 
- 获取并安装

```bash
$ go get github.com/baidu/bfe
```

可执行目标文件位置: ${GOPATH}/bin/bfe

## 运行
- 基于示例配置运行BFE:

```bash
$ cd ${GOPATH}/bin/ 
$ ./bfe -c ${GOPATH}/src/github.com/baidu/bfe/conf/
```
