# 参数说明


## 参数help
- 执行如下命令:

```bash
$ ./bfe -h
```

| 参数     | 类型 |  说明  |
| ------------ | -------- | -------- |
| -c | string | 指定配置文件的目录（默认 "./conf"） |
| -d | bool | 显示debug调试i日志（如不指定默认显示info日志）|
| -h | bool | 显示帮助信息 |
| -l | string| 指定日志路径（默认 "./log"） |
| -s | bool | 将日志信息输出至stdout标准输出 |
| -v | bool | 显示bfe版本 |
