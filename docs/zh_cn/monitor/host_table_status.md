# Host Table Status 

## 简介

host_table_status是路由相关配置的状态。

## 监控项

| 监控项                 | 描述               |
| --------------------- | ------------------ |
| HostTableSize         | 产品线域名表的大小    |
| HostTagTableSize      | 产品线域名标签表的大小 |
| VipTableSize          | 产品线VIP表的大小    |
| ProductRouteTableSize | 产品线路由表的大小    |
