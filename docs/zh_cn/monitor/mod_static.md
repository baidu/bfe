# 简介

mod_static支持静态文件访问。

# 监控项

| 监控项                   | 描述                              |
| ----------------------- | --------------------------------- |
| FILE_BROWSE_COUNT       |统计BROWSE请求数                    |
| FILE_CURRENT_OPENED     |统计当前打开的文件数                  |
| FILE_BROWSE_NOT_EXIST   |文件不存在请求数                     |
| FILE_BROWSE_SIZE        |已处理文件总大小                     |
