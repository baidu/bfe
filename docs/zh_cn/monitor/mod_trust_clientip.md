# 简介

mod_trust_clientip 是模块trust_clientip的状态信息。

# 监控信息

| 监控项                       | 描述                                   |
| ---------------------------- | -------------------------------------- |
| CONN_TOTAL                   | 所有连接数                             |
| CONN_TRUST_CLIENTIP          | 来源于信任地址的连接数                 |
