# 均衡详情 

## 简介

bal_table_status 是保存后端集群table的状态。

## 监控项

| 监控项     | 描述                                                         |
| ---------- | ------------------------------------------------------------ |
| Balancers  | 集群状态，该监控项时map数据，key是集群名称，value是集群的状态信息 |
| BackendNum | 所有集群后端实例总数                                         |

### 集群状态信息

| 监控项      | 描述                                                         |
| ----------- | ------------------------------------------------------------ |
| SubClusters | 子集群状态，该监控项时map数据，key是子集群名称，value是该子集群后端实例的总数 |
| BackendNum  | 所有子集群后端实例总数                                       |
