# 简介

mod_http_code统计HTTP响应状态码。

# 监控项

| 监控项              | 描述                 |
| ------------------- |  ------------------- |
| ALL2_X_X            | 2XX响应数            |
| ALL3_X_X            | 3XX响应数            |
| ALL4_X_X            | 4XX响应数            |
| ALL5_X_X            | 5XX响应数            |
