# 简介

mod_auth_jwt支持JWT身份认证。

# 监控项

| 监控项                   | 描述                                |
| ------------------------ | ---------------------------------- |
| AUTH_TOTAL               | 命中认证规则的请求数                  |
| AUTH_SUCCESS             | 认证成功的请求数                      |
| AUTH_FAILED              | 认证失败的请求数                      |
