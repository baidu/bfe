# Install using binaries

## Download binaries

- Download address: [https://github.com/baidu/bfe/releases](https://github.com/baidu/bfe/releases)

## Extract

- Extract the files to the installation directory:

```
$ tar zxvf bfe.tar.gz
```

## Run

```
$ cd bfe/bin
$ ./bfe -c ../conf -l ../log
```
